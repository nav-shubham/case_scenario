5. URL ROUTING SCHEME (Hash Router)
----------------------------------
Deep link URL routing is implemented using hash states:
- `#/login` -> Candidate Portal login form
- `#/dashboard` -> Level Selection (CA Final & Intermediate)
- `#/dashboard/final` -> CA Final Subject selection list
- `#/dashboard/intermediate` -> CA Intermediate Subject selection list
- `#/workspace/<level>/<subject>` -> Resolves subject, auto-opens first case scenario and first question
- `#/workspace/<level>/<subject>/case/<caseNo>/q/<qIndex>` -> Directly deep links into the specified case number and question index (e.g. `#/workspace/final/AFM/case/1/q/2`)
